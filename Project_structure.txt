================================================================================
ROLE MAP (SHORT)
================================================================================

- run_bot.py / run_bot.sh / run_bot.ps1:
  Launcher chính, preflight, khởi động BotCore + MessageHandler và SearchSubtaskWorker (nếu SEARCH_SUBTASKS_ENABLED=true).

- stop_infra.sh / stop_infra.ps1:
  Teardown hạ tầng (Kafka/Zookeeper/PostgreSQL) theo 2 pha.

- install_services.sh / install_services.ps1:
  Bootstrap Java/Kafka/PostgreSQL project-local trong src/.runtime; Linux có fallback PostgreSQL host/build source.

- src/core/*:
  Router, key pool, proxy, prompt loader, preflight. retrieval_prompts.txt được tách thành 6 prompt con (three_block_context, search_budget, extra_retrieval, partial_evidence, auto_retrieval_limit, final_synthesis).
  gemini_api_manager.py có cầu nối TOOL_CALL cho LM Studio local và guard correction.

- src/handlers/*:
  BotCore nhận sự kiện Discord, gom attachments (current + referenced), gửi Kafka, sanitize mentions, typing loop 5s, và các slash command:
  /ping, /reset-chat, /reset-all, /health_check, /enable_custom_api, /custom_api_config, /imagine, /premium, /moderator, /global-notes, /global-note-demote, /message_to, /donate.
  MessageHandler là worker Kafka, xử lý RAG, ảnh (inline_data), pipeline, và gửi reply_batch (chunk 1800 ký tự).

- src/database/repository.py:
  Asyncpg repo, schema + index GIN/Trgm, custom_api_provider_config (endpoint_preset), custom_api_models, bot_model_config.

- src/services/*:
  KafkaService chung, file parser/indexing, health checker, search subtask client/worker.

- src/tools/tools.py:
  Tool registry gồm web_search, get_weather, calculate, save_note, retrieve_notes, delete_note, image_recognition, manage_user_role (Admin-only schema). Search pipeline hỗ trợ temporal, multi-intent, rerank, cache, dynamic timeout; image_recognition dùng Gemini Flash (GenAI SDK).

- src/managers/*:
  CacheManager (TTL/LRU), NoteManager (tóm tắt LLM Lite + promote global), PremiumManager (check/add/remove).

- src/voice/voice_lock.py:
  Voice lock và whitelist quản lý phòng thoại; BotCore đăng ký /lock, /unlock, /move, /move_all, /set_room, /add_privet, /remove_privet, /list_privet.
