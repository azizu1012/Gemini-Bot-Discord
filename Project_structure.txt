================================================================================
ROLE MAP (BẢN ĐỒ VAI TRÒ CHÍNH)
================================================================================

1. LAUNCHER & LIFECYCLE (run_bot.py, install_services.py, stop_infra.py):
   Đảm nhận toàn bộ vòng đời của dự án. Tự khởi tạo môi trường, tải hạ tầng cục bộ, kiểm tra kết nối (Preflight check), dọn dẹp tiến trình cũ bị kẹt trước khi khởi chạy và dọn sạch hạ tầng khi kết thúc.

2. API GATEWAY (src/handlers/bot_core.py):
   Giao diện kết nối trực tiếp với Discord API. Tiếp nhận tin nhắn và Slash Command, thực hiện kiểm tra spam nhanh, chạy task gửi Typing Status liên tục mỗi 5s, đóng gói payload JSON và đẩy vào Kafka topic "discord-incoming". Khi nhận phản hồi từ topic "discord-outgoing", Gateway điều phối hàng đợi gửi tin tuần tự để tránh rate limit của Discord.

3. DISTRIBUTED WORKER (src/handlers/message_handler.py):
   Tiến trình xử lý nền chính. Kéo sự kiện từ Kafka, tải và phân tích file đính kèm, tải ảnh lên RAM tạm thời để nhận diện (RAM-Only), truy vấn nhanh RAM KV Cache để lấy lịch sử chat, kích hoạt pipeline lập luận Gemini, gọi công cụ và đẩy kết quả phản hồi về Kafka topic "discord-outgoing".

4. BUSINESS LOGIC & MANAGERS (src/managers/*, src/services/*):
   Cung cấp các dịch vụ phụ trợ như tóm tắt ghi chú thông minh (NoteManager), quản lý bộ nhớ RAM siêu tốc (CacheManager), lập chỉ mục tài liệu (FileIndexService), và giám sát sức khỏe API keys ngầm (HealthChecker).

5. DATABASE LAYER (src/database/repository.py):
   Thực thi các truy vấn PostgreSQL tham số hóa an toàn tuyệt đối chống SQL Injection. Tự động đăng ký codecs asyncpg để đọc/ghi trực tiếp Python dict/list vào cột JSONB, tận dụng tối đa chỉ mục GIN để tăng tốc độ truy vấn ghi chú lên gấp 50 lần.
