================================================================================
ROLE MAP
================================================================================

Root files:
- main.py:
  Thin entrypoint gọi launcher chính.

- run_bot.py:
  Runtime launcher chính (bot-only hoặc bot + Flask server), log thông tin startup quan trọng.

- run_bot.sh:
  Launcher Linux/Ubuntu, hỗ trợ PM2 mode (`--pm2`) và PM2 fresh reinstall (`--pm2-fresh`).

- README.md:
  Quick start + kiến trúc ngắn gọn + command/tool overview.

- PROJECT_INFO.txt:
  Technical overview chi tiết hơn, gồm memory architecture và roadmap.

- AGENT_HANDOFF_PROJECT_CONTEXT.md:
  Context gói gọn để gửi cho agent phân tích ngoài khi không thể gửi full source.

Core runtime modules:
- src/handlers/bot_core.py:
  Khởi tạo bot, đăng ký slash commands, quản lý admin interaction views.

- src/handlers/message_handler.py:
  Routing message, build context, intent detection, file indexing orchestration, gọi pipeline. Chống duplicate spam via `processing_users` lock.

- src/core/gemini_api_manager.py:
  API key pool rotation, quota management, cooldown, client pool, throttling, error classification. Bao gồm CustomAPIWrapperResponse cho tương thích endpoint OpenAI.

- src/core/gemini_pipeline.py:
  Two-tier pipeline (reasoning loop → final synthesis → fallback), evidence evaluation, artifact cleaning. Mọi truy cập vào response.candidates sử dụng wrapper an toàn.

File indexing:
- src/services/file_index_service.py:
  LLM-powered file indexing (chunk → reasoning index → Flash validation). Helper functions ngoài class.

- src/services/file_parser.py:
  Download, chunk, text extraction cho file đính kèm Discord.

Memory and persistence:
- src/database/repository.py:
  SQLite schema bootstrap/rebuild compatibility path + queries + global note moderation operations.

- src/managers/note_manager.py:
  Classification policy cho notes, retrieval scope logic, promote candidate->global.

- src/services/memory_service.py:
  Legacy JSON memory utility path.

Tool layer:
- src/tools/tools.py:
  Khai báo tools, dispatch tool-calls, và lớp ổn định web_search (cache/dedup/cooldown/deep-read).

Prompt and config:
- src/instructions/system_prompt.py, lite_sys_prompt.py, fallback_sys_prompt.py:
  Python fallback constants cho prompt templates (dùng khi file .txt không tồn tại).

- src/core/prompt_loader.py:
  LRU-cached loader cho tất cả prompt templates từ src/instructions/*.txt.
  Có Python fallback constants khi file không tồn tại.

- src/instructions/:
  Tất cả prompt templates dùng cho pipeline (system, reasoning, fallback, identity, file index, etc.).

NOTES
